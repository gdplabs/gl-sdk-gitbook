# Generative AI First Quest

Hi, AI engineers! Welcome to the Generative AI team!

This repository contains resources needed in order to complete your first quest tasks as part of the Generative AI team.
The resources are as follows:
1. `data/imaginary_animals.csv`: A CSV file containing dataset to be indexed to a vector database.
The data will be used as a source of knowledge for the RAG pipeline that you're going to build in your first quest.
2. `.pre-commit-config.yaml`: The pre-commit hook configurations.
3. `docker-compose.yml`: The Docker Compose configuration for a local Elasticsearch container.
4. `indexer.py`: A Python script that indexes the provided CSV data into a vector database.
5. `pyproject.toml`: The Poetry configurations for the first quest project.
6. `README.md`: The README file of the Generative AI first quest Github repository.

For more information about the first quest as well as how to utilize these resources, please refer to the [GLAIR Gen AI AIE Onboarding](https://docs.google.com/document/d/1hcfuBcZbF0Bl_XHR7TSYybDxU_Pf61R8kCLAcQhj9IY/edit) document.

Have fun and enjoy the process!
