## eKYC

eKYC refers to the process of electronically verifying the identity of an individual or entity. This involves using various forms of digital identification, such as:

* Biometric data (e.g., facial recognition)
* Government-issued IDs (e.g., KTP)

To help you automate eKYC, we provide several API endpoints, including:

* **OCR** (e.g., KTP, NPWP)
* **Face Biometrics** (e.g., Face Matching)

## Paperless

Paperless refers to conducting business or personal transactions without using physical paper. This involves digitizing documents and data so they can be:

* Stored electronically
* Accessed easily
* Shared efficiently
