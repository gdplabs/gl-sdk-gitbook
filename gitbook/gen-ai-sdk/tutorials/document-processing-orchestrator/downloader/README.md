# Downloader

## What is Downloader?

**Downloader** is designed for **downloading resources from a given source** and save the output into a file.

Our Downloader has the following sub components to handle various types of sources:

1. [Direct File URL Downloader](direct-file-url-downloader.md)
2. [Google Drive Downloader](google-drive-downloader.md)
3. [HTML Downloader](html-downloader.md)
4. [Firecrawl Downloader](firecrawl-downloader.md)

## API Reference

For more information about Downloader, please take a look at our [API Reference page](https://api.python.docs.gdplabs.id/gen-ai/library/gllm_docproc/api/downloader.html).
