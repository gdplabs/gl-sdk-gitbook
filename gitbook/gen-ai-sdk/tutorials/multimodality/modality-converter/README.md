# Modality Converter

### What's a Modality Converter?

Modality Converter is component responsible for converting input from a modality to other modalities.

Currently, the component supports this modality conversion:

1. [Image to Text](image-to-text-converter/)
2. [Audio to Text](audio-to-text-converter.md)
3. [Video to Text](video-to-caption.md)
