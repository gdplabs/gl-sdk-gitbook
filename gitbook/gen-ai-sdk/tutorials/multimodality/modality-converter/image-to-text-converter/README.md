# Image to Text Converter

### What's an Image to Text Converter?

**Image to Text** **Converter** is designed to **convert images into various text format** for multimodal Gen AI applications, enabling text-based LLMs to process and understand visual data.

Our Image to Text converter has the following sub components to handle various types of image to text conversion:

1. [Image to Caption](image-to-caption.md)
2. [Image to Mermaid](image-to-mermaid.md)
