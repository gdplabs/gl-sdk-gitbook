# Image Modality Transformer

### What's an Image Modality Transformer?

**Image Modality Transformer** is a wrapper component designed to **transform images input into various format** for multimodal Gen AI applications.

Our Image Modality Transformer has the following sub components to handle various types of image transformation:

1. [Standard Image Modality Transformer](standard-image-modality-transformer.md)
2. [Generic Image Modality Transformer](generic-image-modality-transformer.md)
