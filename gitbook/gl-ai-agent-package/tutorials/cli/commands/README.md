CLI commands for managing accounts, agents, tools, MCPs, models, and runs.

## Commands

| Command | Purpose |
| --- | --- |
| [Accounts](accounts.md) | Manage credential profiles and switch between environments. |
| [Status](status.md) | Validate connectivity and environment health. |
| [Agents](agents.md) | List, inspect, create, and run agents. |
| [Tools](tools.md) | Attach and manage tools for agents. |
| [MCPs](mcps.md) | Configure and manage MCP servers. |
| [Models](models.md) | List and inspect language models. |
| [Runs and Transcripts](runs-and-transcripts.md) | Capture and view run output. |
