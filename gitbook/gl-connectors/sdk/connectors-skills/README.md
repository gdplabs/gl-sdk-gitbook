# Skills

## Overview

GL Connectors Skills is a subset of GL Connectors delivered as an SDK, designed to easily and programmatically install **Agentic Skills** into your applications.

## What are Agent Skills?

Agent Skills are folders of instructions, scripts, and resources that AI agents can discover and use to perform tasks more accurately and efficiently. They provide agents with:

* **Procedural knowledge** — step-by-step instructions for complex tasks
* **Contextual awareness** — company-, team-, and user-specific information
* **On-demand capabilities** — skills loaded dynamically based on the task at hand

## Key Benefits

| Benefit                  | Description                                                                          |
| ------------------------ | ------------------------------------------------------------------------------------ |
| **Domain Expertise**     | Package specialized knowledge into reusable instructions                             |
| **New Capabilities**     | Enable agents to create presentations, build MCP servers, analyze datasets, and more |
| **Repeatable Workflows** | Turn multi-step tasks into consistent, auditable processes                           |
| **Interoperability**     | Reuse the same skill across different skills-compatible agent products               |

## Why Use GL Connectors Skills?

* **Programmatic Installation** — Integrate agentic skills into your system via SDK
* **Standardized Format** — Built on the open Agent Skills specification originally developed by Anthropic
* **Portable & Version-Controlled** — Capture organizational knowledge in maintainable packages
