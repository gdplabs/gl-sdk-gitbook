# Prerequisites

Before you begin using GL Open DeepResearch, ensure your environment is ready.

{% stepper %}
{% step %}
**Python 3.11+**

Make sure you have [Python](https://www.python.org/downloads/) v3.11 or v3.12 installed along with [pip](https://pip.pypa.io/en/stable/installation/).

```bash
python --version

# Python 3.11.x or Python 3.12.x
```
{% endstep %}

{% step %}
**GL Open DeepResearch API Key**

Contact GL Open DeepResearch team to obtain API key. Afterwards, you can set it as an environment variable.
{% endstep %}
{% endstepper %}
