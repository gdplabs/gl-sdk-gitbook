# Quick Start

Follow the [Python SDK Quick Start](./python-sdk.md) guide for a blank-folder
setup that covers deploy, run, trigger, streaming, and event history.

Key steps:

1. **Install** — `pip install gl-runner-sdk`
2. **Create a bundle** — `bundles/hello-world/` with `entrypoint.py` + `runnables.yaml`
3. **Deploy** — `runnable.deploy()`
4. **Run** — `runnable.run(payload={"question": "status"})`
