# Prerequisites

Before you start:

- Python 3.11+
- A GL Runner base URL (local or remote)
- An API key with access to your runner environment

Set environment variables (or pass values directly in code/CLI):

```bash
export GL_RUNNER_BASE_URL="http://localhost:4200"
export GL_RUNNER_API_KEY="glr_..."
```

If you are working from this repo, copy the sample env files:

```bash
cp .env.example .env
cp .env.test.example .env.test
```
