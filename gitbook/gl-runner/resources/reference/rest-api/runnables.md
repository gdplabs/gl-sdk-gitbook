# Runnables API

- `POST /v1/runnables:deploy`
- `GET /v1/runnables`
- `GET /v1/runnables/{runnable_id}`
- `POST /v1/runnables/{runnable_id}:run`
- `PATCH /v1/runnables/{runnable_id}` - update runnable metadata/config
- `POST /v1/runnables/{runnable_id}:deploy` - deploy a new bundle version
- `DELETE /v1/runnables/{runnable_id}`
