# Guides

Task-oriented guides for integrating with and operating GL Smart Crawl.

* [**Data API**](data-api.md) — Query crawled articles and property listings with pagination, filtering, and source integrity metadata.
* [**Auto Crawl Scheduler**](auto-crawl-scheduler.md) — Configure and operate the automated per-source crawl scheduler for listing and scraping.
* [**Article Crawl**](article-crawl.md) — Run article listing and scraping with simple, copy-pasteable API calls.
* [**Property Crawl**](property-crawl.md) — Run property listing and scraping with listing type and property type examples.
* [**General Scrape**](general-scrape.md) — Scrape a single URL on demand, including general extraction with `response_type`, custom `schema`, and `prompt`.
