# Architecture Diagrams

High-level architecture diagrams for GL Smart Crawl with focus on the Orchestration Layer boundary.

## Diagrams

* [**Block Diagram**](block-diagram.md) — System-level view of GL Smart Crawl (Orchestration + Extraction Layers), GL Smart Search, and their connections

## Usage

All diagrams use [Mermaid](https://mermaid.js.org/) syntax and can be rendered in:
- GitBook (natively supported)
- GitHub Markdown
- VS Code with the Mermaid extension
- Any Mermaid-compatible viewer

## Updating Diagrams

Keep diagrams in sync with code changes. When modifying the service architecture, update the corresponding diagram in this directory.
