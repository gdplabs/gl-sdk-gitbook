# Guides

This section provides step-by-step guides for integrating and using GL Smart Search capabilities.

Choose a guide based on your integration method:

* [**Authentication**](authentication.md) — Learn how to get your authentication token to access GL Smart Search.
* [**Software Development Kit (SDK)**](sdk/) — Use the Python SDK for web search and connector search with full code examples.
* [**Model Context Protocol (MCP)**](mcp.md) — Integrate GL Smart Search as an MCP tool into agentic systems and AI assistants.
* [**Command Line Interface (CLI)**](cli/) — Use GL Smart Search directly from your terminal with CLI commands.
