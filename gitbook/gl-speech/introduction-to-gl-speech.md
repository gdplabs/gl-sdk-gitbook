# Introduction to GL Speech

**GL Speech** is a unified speech processing platform that provides Speech-to-Text (STT), Text-to-Speech (TTS), and Webhook management capabilities. Users can leverage various speech processing features through a single, consistent API interface without needing to manage individual services or their configurations.
